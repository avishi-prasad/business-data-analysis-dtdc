# DTDC Business Data Analysis Project

## Overview
This project focuses on analyzing operational and revenue-related data for a **DTDC courier franchise** to derive actionable business insights. The analysis aims to understand performance trends, identify inefficiencies, and support data-driven decision-making.

---

## Objectives
- Identify revenue trends across delivery regions to optimize resource allocation and improve operational planning.
- Analyse return rates and reasons to minimize parcel returns and enhance product/service quality.   
- Optimize delivery routes to reduce delays, lower operational costs, and improve customer satisfaction.  

---

## Tools & Technologies
- **Excel** – Data cleaning, transformation, and analysis  
- **Power BI** – Interactive dashboards and visualizations
- **Python** - OpenRouteService API  
- **Basic Statistics & Business Metrics**
